# Ithi-Sujith-Babu-Project-Full-Stack-Developer-Position-at-TANDEMLOOP

##This project contains four Java programs demonstrating basic programming concepts: arithmetic operations, series generation, and counting multiples. ##

## Problem 1 – Simple Calculator

## Description
A calculator program that performs basic arithmetic operations: addition, subtraction, multiplication, and division.  

## Inputs
- `a` : double  
- `b` : double  
- `operation` : String (`+`, `-`, `*`, `/`)  

## Sample Input
   Enter a: 10
   Enter b: 5
   Enter operation (+, -, *, /): *
## Result: 50.0

## Problem 2 – Odd Number Series

## Description
Generates the first `a` odd numbers starting from 1.

## Input
- `a` : integer (number of terms)

## Sample Input / Output
   Input: 4
   Output: 1, 3, 5, 7
<---------------------------------------------------------------------------------------------->

Problem 3 – Conditional Odd Series

## Description
Generates odd numbers up to `a`. The series includes all odd numbers less than `a`, and if `a` is odd, it is included as well.

## Input
- `a` : integer

## Sample Input / Output

Input: 6
Output: 1, 3, 5

Input: 5
Output: 1, 3, 5

<----------------------------------------------------------------------------------------------->

## Problem 4 – Count Multiples in Array

## Description
Counts how many numbers in a given array are multiples of 1 through 9.  

## Input
   [1, 2, 8, 9, 12, 46, 76, 82, 15, 20, 30]
## Output
   {1=11, 2=8, 3=4, 4=4, 5=3, 6=2, 7=0, 8=1, 9=1}
